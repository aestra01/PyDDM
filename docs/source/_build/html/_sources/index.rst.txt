Welcome to DDM_py's documentation!
=====================================================

**DDM_py**  is a Pyhton class based framework for analysis of differential dynamic microscopy (DDM) data. 

The analysis is performed in two steps:
	1.DDM analysis to calculate the DDM matrix and ISF
	2.Fitting with multiple mathematical models
	


.. note::

   This project is under active development.
